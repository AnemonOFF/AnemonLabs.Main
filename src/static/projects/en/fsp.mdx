**FSP** - Federation of Sports Programming of the Sverdlovsk Region is a regional branch of an all-Russian public organisation that is interested in developing a portal for keeping records and statistics of events, participants, as well as functionality for partners for further development.

My role in this project is front-end developer. The team uses a Scrum methodology. The code is stored in a common repository, each person working on their part in separate branches.
