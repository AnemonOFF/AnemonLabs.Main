**DotStat** is a web application designed to provide comprehensive, up-to-date information on residential complexes with available flats, pantries, parking spaces, and commercial areas. The back end of **DotStat** includes an API and a robust data-parsing system that continuously gathers and organizes listings of properties currently for sale.

In the web interface, users can browse through all collected residential complexes, view essential information, and easily download detailed reports in XLSX format for further analysis. This setup allows users to stay informed on real estate options with minimal effort, making **DotStat** an efficient tool for anyone needing quick access to the latest housing and commercial property data.
