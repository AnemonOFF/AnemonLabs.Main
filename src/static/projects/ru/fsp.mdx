**ФСП** - Федерация спортивного программирования Свердловской области - региональное отделение всероссийской общественной организации, у которой есть заинтересованность в разработке портала для ведения учета и статистики мероприятий, участников, а также функционала для партнеров для дальнейшего развития.

Моя роль в этом проекте - Front-End разработчик. Команда работает по Scrum методологии. Код хранится в общем репозитории, каждый участник работает над своей частью в отдельных ветках.
