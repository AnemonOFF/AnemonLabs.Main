**FontNET** - это AI платформа, которая обеспечивает перевод пользовательских шрифтов с латиницы на кириллицу. Разработанная с учетом требований удобства использования, **FontNET** служит дизайнерам, лингвистам и создателям контента, которым необходимо адаптировать шрифты для многоязычной аудитории, сохраняя при этом их уникальные стилистические элементы.

В этом проекте я работал в сотрудничестве с командой, состоящей из двух инженеров по искусственному интеллекту и руководителя группы, используя Git для контроля версий и придерживаясь Agile-методологии для эффективного управления рабочим процессом и итерациями. В мои основные обязанности входила разработка внешнего приложения, создание интуитивно понятного интерфейса, позволяющего пользователям изучать возможности **FontNET**, загружать пользовательские шрифты и получать переведенные версии после обработки. Внешний интерфейс был создан с акцентом на простоту и удобство для пользователей, обеспечивая упрощенный путь от загрузки до скачивания.

Для поддержки функциональности между фронт-эндом и моделью искусственного интеллекта я также разработал промежуточный API. Этот API обрабатывает запросы пользователей, управляет связью между фронт-эндом и моделью искусственного интеллекта и реализует механизм кэширования для хранения ранее переведенных шрифтов, сокращая время обработки для повторных пользователей и оптимизируя общий пользовательский опыт.
